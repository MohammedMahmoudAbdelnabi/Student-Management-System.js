# Student Management System 📚

A simple JavaScript console project that manages students and their grades using arrays and loops.

## 🚀 Features

- Display students and their grades
- Add a new student
- Search for a student
- Calculate the average grade
- Find the highest grade
- Find the lowest grade

## 🧠 Concepts Used

This project practices important JavaScript fundamentals:

- Arrays
- For Loops
- If Conditions
- Array Methods (push)
- Variables
- Basic Algorithms
