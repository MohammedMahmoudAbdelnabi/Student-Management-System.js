# Student Management System 📚

A simple JavaScript project that manages **students and their grades** using arrays.
The project demonstrates how to store related data in arrays, loop through them, search for specific students, and perform basic calculations like average, highest, and lowest grades.

This project is part of my JavaScript learning journey and focuses on building logical thinking using arrays and loops.

---

## 📌 Features

* Display all students with their grades
* Add a new student with a grade
* Search for a student by name
* Calculate the **average grade**
* Find the **highest grade**
* Find the **lowest grade**

---

## 🧠 Concepts Used

This project practices important JavaScript fundamentals:

* JavaScript Arrays
* Array Indexing
* Loops (`for`)
* Conditional Statements (`if`)
* Array Methods (`push`)
* Basic Arithmetic Operations
* Console Output
